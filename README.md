## commit 및 push 방법

- 본인의 이름(닉네임)으로 된 브랜치를 판다.
- 각 장의 폴더에 본인의 이름(닉네임)으로 폴더를 만들고, 정리한 내용을 .md파일로 올린다.
  - ex) 01\_데이터타입 > leedawn > 01_summary.md
- 본인 브랜치 push
- pr 날리기
  - 본인의 브랜치 -> main

## 터미널 명령어

1. git fetch upstream main
2. git pull upstream main
3. git add .
4. git commit -m "`이름`: `내용`"
5. git push origin `본인 브랜치명`
